# Vue 3 + Vite

Установка и запуск проекта

1. Установите зависимости
   Сначала откройте терминал и выполните команду:

npm install

2. Запустите проект в режиме разработки
   После установки зависимостей, запустите проект с помощью команды: npm run dev Теперь вы сможете открыть проект в браузере по адресу http://localhost:4173/ (или другому, указанному в терминале).

3. Подготовка к продакшену
   Если вы хотите собрать проект для продакшена, выполните команду: npm run build Это создаст оптимизированную версию вашего приложения.

4. Посмотреть превью проекта после сборки, выполните команду: npm run preview
   Теперь вы сможете открыть проект в браузере по адресу http://localhost:4173/ (или другому, указанному в терминале).
