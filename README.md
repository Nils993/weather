Проект: Веб-приложение погоды
Суть проекта: Создание интерактивного одностраничного приложения для отображения текущей погоды по введённому городу.
Что удалось сделать:
•	Разработана пользовательская форма поиска с двунаправленным связыванием данных (v-model) для ввода названия города
•	Реализована логика получения данных о погоде через API с использованием fetch, обработка загрузки и ошибок
•	Созданы компоненты для отображения информации о местоположении, температуре и описании погоды
•	Реализована динамическая смена фона сайта в зависимости от текущих погодных условий с помощью вычисляемого свойства (computed)
•	Настроена стилизация интерфейса с использованием CSS3

Стек: Vue.js, Vite, JavaScript, API ,HTML, CSS 

Установка и запуск проекта
1. Установите зависимости
   Сначала откройте терминал и выполните команду:
npm install
2. Запустите проект в режиме разработки
   После установки зависимостей, запустите проект с помощью команды:
npm run dev
   Теперь вы сможете открыть проект в браузере по адресу http://localhost:4173/ (или другому, указанному в терминале).

4. Подготовка к продакшену
   Если вы хотите собрать проект для продакшена, выполните команду: npm run build Это создаст оптимизированную версию вашего приложения.

5. Посмотреть превью проекта после сборки, выполните команду: npm run preview
   Теперь вы сможете открыть проект в браузере по адресу http://localhost:4173/ (или другому, указанному в терминале).
